Git Important Commands

1. which git ( To check git installed or Not)

2. git --version ( To check the Git Version)

3. git init ( initilize the directory with git local repo)

4. git status ( To Check the Git Status )

5. git add . / git add filename ( Add the file or Code in Got repo)

6. git commit -m "typemassage" ( Commit the file or Code in Git local Repo)

7. git log / git log --oneline (To Check the Git commit logs)

8. git show commitid ( To check the Code in details with commit id)

9. git config --global user.name "username" ( Configure the user name for git)

10.git config --global user.email "useremailid" (Configure the Email id for ID)

11. git remote add origin (Centeral repo URL) ( Connect with Centeral Repo)

12. git push origin master (BranchName) ( Push the code in Centeral Repo)

13. git branch (Check the branch list)

14. git branch branchname (to create a branch)

14. git checkout branchname ( to change the branch)

15. git branch -d branchname ( to Delete the banrch)

16. git merge branchname ( to merge the multiple branch in master branch)

17. git clean -n (dry delete)

18. git clean -f (Forcefully delete)

19. git stash -u ( to add files in stash storage)

20. git stash list ( to check the list of files in stash)

21. git stash apply stashfilename (stash@{0}) to push back in working space)

22. git stash clear ( to clear the stash list)

23. git tag -a tagname -m "write any massage" commit id ( to configure the tag on commit id)

24. git tag ( to check the tag list)

25. git show tagname ( to check the data/code with tagname in commid id)

26. git tag -d tagname ( to Delet the tag name from tag list)

27. git revert commitid (wrong code commit id)

28. git clone Centeral repo URL
